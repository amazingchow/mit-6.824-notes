# MIT 6.824 Distributed Systems

`MIT 6.824 Distributed Systems`是一门讲授分布式系统工程的课程, 主要讲授分布式系统抽象方式和实现技术, 包括但不限于的主题有, 容错(fault tolerance), 副本策略(replication policy), 及一致性(consistency)等.
